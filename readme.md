# Symfony 5 HTTP Client PHPUnit

Proyecto de ejemplo utilizando:

- Symfony HTTP Client [(Link)](https://symfony.com/doc/current/http_client.html)
- Symfony Testing [(Link)](https://symfony.com/doc/current/testing.html)

## Requerimientos
PHP 7.4.26

## Comandos

Composer

```bash
composer install
```

Servidor Symfony 5
```bash
symfony serve:start
```

Pruebas Symfony
```bash
php bin/phpunit
```
